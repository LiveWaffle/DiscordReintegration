# DiscordReintegration
Discord Messages into GregTech!

## Features

- Discord > Minecraft chat
- Minecraft > Discord chat
- Player join messages
- Player leave messages
- Death messages
- Achievement messages
- Server start notifications
- Server stop notifications
- Discord embeds with player avatars
- Configurable Discord channel support




## Supported Versions
- Minecraft 1.7.10
- Forge 10.13.4.1614
- Java 8

## Installation

#### Clone the Repository

```bash
git clone https://github.com/LiveWaffle/DiscordReintegration.git
```
#### Build the Mod
```./gradlew build```
